## Installation

First clone the repo:
```
git clone https://github.com/Mystic2103/TARO-Discord-Music-Bot.git
```
After cloning, run:
```
npm install
```
To launch, run:
```
node .
```

### Emojis & Bot Token
- You can change the emojis and bot token in: <br>
`config.json` <br>
